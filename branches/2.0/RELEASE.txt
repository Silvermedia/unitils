Maven release tasks:

mvn release:prepare
mvn release:perform
mvn release:clean


Step 1: Actual run  	

	check whether everything is ready to be release
	update version 
	create release tag

	mvn release:prepare
	

Step 2: Create release 

	mvn release:perform	

	download clean release
	build unitils 	
	create distribution
	upload the unitils jar to the repository on SourceForge

<settings>
	<servers>
      	<server>
            	<id>sourceforge-repository</id>
                  <username>your_username</username>
            </server>
	</servers>
</settings>
