Maven release tasks:

mvn release:prepare -DdryRun=true
mvn release:prepare
mvn release:perform



Step 1: Dry run  	
	
	mvn release:prepare -DdryRun=true

	check whether everything is ready to be release
	update version 
	create release tag

Step 2: Actual run  	

	mvn release:prepare
	

Step 3: Create release 

	mvn release:perform	

	download clean release
	build unitils 	
	create distribution
	upload the unitils jar to the repository on SourceForge 

Note: to set-up the FTP connection to SourceForge, you need to add following settings.xml to <user.home>/.m2

<settings>
	<servers>
      	<server>
            	<id>sourceforge-repository</id>
                  <username>your_username</username>
            </server>
	</servers>
</settings>
